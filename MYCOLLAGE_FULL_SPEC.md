## 1. プロジェクト概要

### コンセプト
あらゆるWebサイト（Twitter/X, YouTube, TikTok, Instagram, 一般Webサイト等）のブックマークを、
**服飾学生のコラージュ / ムードボード**のような美しいビジュアルボードとして管理・フォルダ分け・共有できるWebアプリ。

### ミッション
「整理ツール」ではなく **「表現ツール」**。
コラージュを画像にしてSNSでシェアさせるバイラルループで成長する。
超モダンで目が楽しい、アニメーションとインタラクションが命のアプリ。

### ターゲット
- **世界中の全ユーザー**（ターゲットは絞らない）
- 特にブックマーク機能が弱いプラットフォーム（X, YouTube等）のユーザー
- 多言語対応で初日から世界展開

## 3. 技術スタック

| 役割 | 選定 | 備考 |

## 5. ユニバーサルブックマーク設計

### URL情報取得フロー

```
ユーザーがURLを保存
    ↓
URLパターン判定
    ├── x.com/twitter.com → react-tweet でツイート表示
    ├── youtube.com        → oEmbed → 動画プレーヤー埋め込み
    ├── tiktok.com         → oEmbed → 動画プレーヤー埋め込み
    ├── instagram.com      → oEmbed → 投稿埋め込み
    └── その他全URL        → OGPスクレイパーWorker
                              → タイトル・説明文・サムネイル・ファビコン取得
                              → カード生成 → コラージュに配置
```

全て無料（APIキー不要）。OGP/oEmbedは各プラットフォーム公式の無料機能。

### フォルダ構造

```
ユーザーのコラージュ
├── 📁 ファッション（カスタムフォルダ）
│   ├── 🐦 @stylist のツイート
│   ├── 🎥 TikTok 動画
│   └── 🌐 ZARA の新作ページ
├── 📁 テック（カスタムフォルダ）
│   ├── 📺 YouTube チュートリアル
│   └── 🌐 tech blog 記事
└── 📁 推し活（カスタムフォルダ）
    ├── 🐦 推しのツイート
    └── 📸 Instagram の投稿
```

各フォルダ = 1つのコラージュキャンバス。

### ブックマークレット

ページ上にフォルダ選択ポップアップを表示する方式:

```
ユーザーがブックマークレットをクリック
    ↓
現在のページ上に小さなポップアップ
    ┌───────────────────────┐
    │  Booklage に保存      │
    │  📁 ファッション       │
    │  📁 テック             │
    │  📁 料理               │
    │  + 新しいフォルダ       │
    │     [保存✨]            │
    └───────────────────────┘
    ↓
保存完了アニメーション → ポップアップ消える
ユーザーはそのページに留まったまま
```

## 8. データ保持方針: 「持たない経営」

- **ユーザー登録なし**
- **データはブラウザ内（IndexedDB）にのみ保存**
- **サーバーにユーザーデータを送らない**
- **GDPR/個人情報保護法の対応がほぼ不要**
- **共有時のみ**: ユーザーが選択 → スナップショットをR2に一時保存（90日自動削除）
- **個人情報は一切含まない**（コラージュ配置情報とURL一覧のみ）

### IndexedDB スキーマ

```typescript
interface AppDB {
  bookmarks: {
    key: string              // UUID
    value: {
      url: string            // 保存したURL
      title: string          // ページタイトル
      description: string    // 説明文
      thumbnail: string      // サムネイル画像URL
      favicon: string        // ファビコンURL
      siteName: string       // サイト名
      type: 'tweet' | 'youtube' | 'tiktok' | 'instagram' | 'website'
      savedAt: Date
      folderId: string
    }
    indexes: { 'by-folder': string; 'by-date': Date }
  }
  folders: {
    key: string              // UUID
    value: {
      name: string           // フォルダ名
      color: string          // テーマカラー
      createdAt: Date
      order: number
    }
  }
  cards: {
    key: string              // UUID
    value: {
      bookmarkId: string     // 対応するブックマークのID
      folderId: string       // 所属フォルダ
      x: number              // X座標
      y: number              // Y座標
      rotation: number       // 回転角度（度）
      scale: number          // 拡大率
      zIndex: number         // 重なり順
    }
    indexes: { 'by-folder': string }
  }
  settings: {
    key: string
    value: {
      theme: string          // 背景テーマ
      locale: string         // 表示言語
      customImage: Blob | null  // カスタム背景画像
    }
  }
}
```

### アフィリエイト: コンテンツ連動型表示

- ユーザーのコラージュ内容（フォルダ名・ブックマークのカテゴリ）に基づき、関連する商品リンクを表示する。
- 例: ファッション系のコラージュ → ファッションアイテムのアフィリエイト、テック系 → ガジェット系
- 表示はコラージュのデザインを壊さない**おしゃれなカード形式**で、他のブックマークカードと馴染むようにする。
- アフィリエイトカードであることは明示する（「AD」ラベル + 透明性確保）。
- Amazonアソシエイト + 楽天アフィリエイトは公開後すぐに申請・導入する。

## 15. 実装・運用ガイドライン（2026-04-10 追加）

### コア・コンセプト: No Data Collection

- 「一切のデータをサーバーに保存しない（No Data Collection）」を最大の哲学とする。
- プライバシー重視のユーザーが、コードを見て安心できる設計（オープンソース）を徹底する。
- README.md に「なぜデータを持たないのか」という想いを日英両方で記載する。

### GitHub公開と透明性の維持

- 本プロジェクトは**最初からGitHub Publicで開発**する。
- `.env` 管理を徹底し、秘密情報が混入しないよう常に監視する。
- `.gitignore` に `.env*`, `.dev.vars`, `.superpowers/` を含める。
- コードレビューでシークレット漏洩がないか毎回確認する。

### Drop-zone PiP（将来拡張: v1.2以降）

- Spotify風の浮遊窓（PiP）を実装し、他サイトからのURLドラッグ＆ドロップで保存する体験を構築する。
- **Document Picture-in-Picture API** を使用（Chromium系ブラウザ限定）。
- MVPではブックマークレットが主力。PiPはChrome限定のボーナス機能として後から追加。
- PiPの「ドラッグ＆ドロップの気持ちよさ」はプロモ動画・バイラル素材として最大限活用する。

### モバイル保存体験: Web Share Target API

- PWAとしてインストールすると、OSの共有メニューに「Booklage」が出現する。
- Twitter/YouTube/どのアプリからでも「共有→Booklage→フォルダ選択→保存」で完結。
- デスクトップChromeでもPWAインストール後は共有ボタンから保存可能。

### ブックマークレット: 「魔法の杖」チュートリアル

- ブックマークレットのセットアップは非エンジニアにとって分かりにくいため、**世界一わかりやすいチュートリアル**を用意する。
- **インタラクティブガイド**: 画面上で「ここにドラッグ！」と矢印が動く、ステップバイステップのアニメーション。
- **短い動画チュートリアル**: 15秒以内で完結する動画も併設。
- 両方を組み合わせ、ユーザーが好きな方法でセットアップできるようにする。

### 保存方法の優先順位

| 優先度 | 方法 | 対象 | タイミング |
|--------|------|------|-----------|
| **1** | ブックマークレット | デスクトップ全ブラウザ | MVP |
| **2** | Web Share Target（PWA） | モバイル全般 + デスクトップChrome | MVP |
| **3** | URLペースト | 全デバイス（フォールバック） | MVP |
| **4** | PiPドロップゾーン | Chrome/Edge限定ボーナス | v1.2 |
| **将来** | ブラウザ拡張機能 | 全デスクトップ | 収益化後に検討 |
